日本住宅価格予測モデル

日本全国の住宅価格を予測する教師あり機械学習モデルを開発しました。 物件情報、地理データ、公示地価データを組み合わせることで、 評価精度を向上させ、価格戦略の策定に貢献しました。

Python Pandas scikit-learn NumPy Seaborn Jupyter Notebook Google Colab
Approach: Conducted data cleaning and preparation, selected high-correlation features, and engineered unique features such as district names and building ages. Performed model selection and hyperparameter tuning to maximize performance.
Outcome: Model significantly outperformed a simple baseline with r2 score of 0.58 (median-price predictor), provided interpretable feature importances (land area, proximity to stations, year built etc), and was wrapped into a small API for quick valuation lookups.

AIを活用し、複数のPDFから不動産情報を自動抽出し、CSV形式に変換するツールを開発しました。 変換されたデータは自動的にデータベースにアップロードされ、 クライアントの手作業によるデータ入力の負担を大幅に軽減しました。
